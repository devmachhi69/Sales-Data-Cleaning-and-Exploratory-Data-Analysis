# Sales Data Cleaning and Exploratory Data Analysis

This project focuses on cleaning and analyzing a sales dataset using Python.

## Key Tasks Performed
- Handling missing (null) values
- Treating negative values in quantity
- Detecting and handling outliers using the IQR method
- Standardizing product names and payment methods
- Creating date-based features (year, month)
- Visualizing data using bar charts, boxplots, and pie charts

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Outcome
The dataset was cleaned and transformed into an analysis-ready format, enabling accurate insights and visualizations.
